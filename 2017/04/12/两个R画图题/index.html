1.

请根据depth_cover_barplot.txt文件里的内容作图

说明：

(1) 文本文件第一行是指染色体，第一列是样本名。

(2) pdf文件横坐标是指染色体，左边纵坐标是该染色体的平均覆盖深度。


R代码：

rm(list = ls())
setwd("D:\\test\\test4")
aaa<- read.table("depth_cover_barplot.txt")
pdf("plot.pdf",height = 6,width = 13)
par(mgp=c(1.6,0.6,0),mar=c(3,3,3,1))
barplot(as.matrix(aaa),
        ylim = c(0,400),
        ylab = "Mean Depth",
        main = "Depth of Coverage",
        col = rainbow(3),
        beside = TRUE,
        font = 2, cex.lab = 1.4,
        border = NA
        )
legend("topright",
       legend = c("S1","S2","S3"),
       pch = 15,
       col = rainbow(3),
       bty = "n")
dev.off()

2.

说明：

  1. 请读取base_stat_total.txt文件里的内容

  2. 文本文件第一行代表A,C,G,U四个碱基，其他行代表每个碱基在第18-25个位置上各个碱基所占的比例。

R代码：

rm(list = ls())
setwd("D:\\test\\test5")
aaa<- read.table("base_stat_total.txt",header = TRUE)
row.names(aaa)<-c(18:25)
pdf("plot.pdf",height = 6,width = 8)
par(mgp=c(1.6,0.6,0),mar=c(3,3,5,5),xpd=T)
barplot(t(as.matrix(aaa)),
        ylim = c(0,100),
        ylab = "Percent(100%)",
        xlab = "Lenth",
        main = "total miRNA first nucleotide bias",
        col = rainbow(4),
        font = 1, cex.lab = 1,
        #border = NA
        )
legend("topright",
       legend = c("A","C","G","U"),
       pch = 15,
       col = rainbow(4),
       inset = c(-0.05,0)
       )
dev.off()
