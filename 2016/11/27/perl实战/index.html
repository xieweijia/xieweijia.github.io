1 统计文本中的每个单词出现次数并排序

用Perl语言写一段代码,计算出文本 1.txt 中有多少个单词（重复的也算），并按照 ascii码表顺序输出每个单词在文本中的频率。

要求如下格式化输出：

A                   =>         3
All                 =>         1
Books               =>         3
But                 =>         4
Companionship       =>         1
Darkness            =>         1

代码记录：

#! usr/bin/perl
use strict;
my(@line,$line,@character,@words,$word,$count,%count);
open INPUT,"D:/课题/我的文件/perl脚本实战/1/1.txt";
open OUTPUT,">D:/课题/我的文件/perl脚本实战/1/result.txt";
@line=<INPUT>;
foreach $line(@line){
chomp($line);
$line=~s/\p{Punct}|“|”|‘//g;   #把引号去掉
@character=split /\s+/,$line;  #按照空格分割
push @words,@character;        #将分好的单词加到末尾
}
foreach $word(@words){         #计数
$count{$word}+=1;
}
foreach $word(sort keys %count){#哈希
print OUTPUT "$word\t=>\t$count{$word}\n";
}

2 编写脚本，找出这两条序列中最长的共有序列

以下2条序列：
序列一：AACTGCACGTGCATCGGATGCATCGATCGTGCGAGTAGTCGATCGATCGTAGCTAGCTCAGTCGATCAGCTACCTCGC
序列二：CGTAGCTACGATCGCATCAGCTACCTCCGTTCGAGTCTGCGCAACGCTACGACTATCGACGTCA
编写脚本，找出这两条序列中最长的共有序列。

#! usr/bin/perl
use strict;
my($s1,$s2,$max,$length,$star,$common,$compare,@compare,$common,@common,$maxcom,%len,$len);
open OUTPUT,">D:/课题/我的文件/perl脚本实战/2/result.txt";
$s1="AACTGCACGTGCATCGGATGCATCGATCGTGCGAGTAGTCGATCGATCGTAGCTAGCTCAGTCGATCAGCTACCTCGC";
$s2="CGTAGCTACGATCGCATCAGCTACCTCCGTTCGAGTCTGCGCAACGCTACGACTATCGACGTCA";
$max=length($s2);
for($star=0;$star<=$max;$star++)
{
   for($length=2;$length<=$max-$star;$length++)
   {
     push @compare,substr($s2,$star,$length);

    }
}

     push @common,$compare;

$len{$common}=length($common);
